# Travis-C 示例脚本

简单的示例脚本代码如下：

```
osx_image: xcode9.4
language: objective-c
xcode_workspace: Demo.xcworkspace
xcode_scheme: Demo
before_install:
  - rm -rf Gemfile.lock
  - sudo gem update --system
  - brew install xctool
  - pod install --repo-update
  - pod install --project-directory=XXX --repo-update
script:
  - set -o pipefail
  - pwd
  - carthage bootstrap --platform iOS
  - xcodebuild -workspace Demo.xcworkspace -scheme Demo -sdk iphonesimulator ONLY_ACTIVE_ARCH=NO build-for-testing | xcpretty
notifications:
  webhooks: https://www.yourwebhooksaddress.com
```
